## 一、软件简介

Navicat for MySQL 是一款经典的 MySQL/MariaDB 可视化管理工具，无需复杂的命令行操作，即可实现以下核心功能，是开发、运维人员必备的数据库客户端：

- **数据库连接管理**（支持本地、远程连接）
- **库表设计**（新建、修改、删除数据库/表，配置字段、主键、索引）
- **SQL 查询**（编写、执行、调试SQL语句，支持代码高亮）
- **数据导入导出**（支持Excel、CSV、SQL等多种格式）
- **数据备份与还原**，保障数据安全

## 三、完整安装步骤（核心环节）

### 1. 解压安装包

下载完成后，找到压缩包，右键选择"解压到当前文件夹"，解压后可看到两个核心文件：

- `navicat111_mysql_cs_x64.exe`（主安装程序，大小约15.2MB）
- `PatchNavicat.exe`（补丁工具，大小约52KB）

### 2. 运行安装程序

1. 双击 `navicat111_mysql_cs_x64.exe`，启动安装向导，点击「下一步」；
2. 勾选"我接受许可协议中的条款"，继续点击「下一步」；
3. 点击「浏览」，选择提前准备好的安装目录（示例：`D:\Software\Navicat for MySQL`），点击「下一步」；
4. 开始菜单文件夹保持默认，无需修改，点击「下一步」；
5. 勾选"创建桌面快捷方式"（方便后续启动），点击「安装」；
6. 等待进度条完成（约1-2分钟），安装结束后，**务必先不要运行软件**，直接点击「完成」并关闭窗口。

### 3. 应用补丁（关键步骤）

补丁用于激活软件，步骤如下，务必操作正确：

1. 打开解压后的文件夹，双击运行 `PatchNavicat.exe`；
2. 在弹出的窗口中，点击「浏览」，找到刚才的安装目录，选中 `navicat.exe`（路径示例：`D:\Software\Navicat for MySQL\navicat.exe`）；
3. 选中后点击「确定」，等待补丁执行，无需其他操作；
4. 当弹出提示 `Patch successfully (3 modifieds)!` 时，说明补丁应用成功，关闭补丁窗口即可。

## 四、MySQL 连接测试（验证安装）

安装完成后，通过连接本地MySQL，验证软件是否能正常使用：

### 1. 新建MySQL连接

1. 双击桌面「Navicat for MySQL」快捷方式，启动软件；
2. 点击软件左上角「连接」→「MySQL」，弹出连接配置窗口；
3. 填写连接信息（新手默认配置即可，根据自身MySQL情况修改）：

   | 配置项 | 说明 |
   |--------|------|
   | 连接名 | 自定义（如「LocalMySQL」，方便区分） |
   | 主机名或IP地址 | `localhost`（本地连接，远程连接填写对应IP） |
   | 端口 | `3306`（MySQL默认端口，未修改过则无需改动） |
   | 用户名 | `root`（MySQL默认管理员账号） |
   | 密码 | 填写你安装MySQL时设置的root密码 |

4. 勾选「保存密码」，避免后续每次连接都输入密码，点击「连接测试」。

### 2. 验证连接成功

点击「连接测试」后，若弹出「连接成功」的提示，说明配置无误，点击「确定」，再点击「保存」，即可完成连接创建。

在软件左侧「连接」列表中，双击新建的连接（如「LocalMySQL」），即可展开本地所有数据库，开始后续操作。

### 3. 常见问题处理（新手必看）

#### ❓ 问题1：新建数据库按钮是灰色的，无法点击？

**解决方法**：直接双击左侧的连接名（如「LocalMySQL」），激活连接后，按钮即可正常使用。

#### ❓ 问题2：连接测试失败，提示"无法连接到MySQL服务器"？

**解决方法**：
1. 检查MySQL服务是否已启动（右键「此电脑」→「管理」→「服务和应用程序」→「服务」，找到「MySQL」，确保状态为"正在运行"）；
2. 检查端口、用户名、密码是否填写正确；
3. 若远程连接，需开启MySQL远程访问权限。

## 五、基础功能快速上手

新手入门常用功能，简单操作即可掌握：

### 1. 新建数据库

右键左侧连接名 →「新建数据库」，填写数据库名称，选择字符集（建议「utf8mb4」，支持所有中文和特殊字符），排序规则默认，点击「确定」。

### 2. 新建表

选中新建的数据库 → 右键「表」→「新建表」，填写字段名、字段类型、长度，设置主键（必选），点击「保存」，命名表名即可。

### 3. 执行SQL语句

点击顶部「查询」→「新建查询」，选择对应的数据库，编写SQL语句（如 `select * from 表名;`），点击「运行」按钮，即可查看执行结果。

### 4. 数据导入导出

选中表或数据库 → 右键 →「导入向导」/「导出向导」，选择对应格式（Excel、CSV等），按提示一步步操作即可。
